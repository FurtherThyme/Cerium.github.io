链接样式用法：

a:link 定义正常链接的样式
a:visited 定义已访问过链接的样式
a:hover 定义鼠标悬浮在链接上时的样式
a:active 定义鼠标点击链接时的样式

如果有些样式相同 也可以合并定义 如

a:link,
a:visited {
...
...
}

书写顺序 link–visited–hover-active LoVe HAte原则（大写字母就是它们的首字母） ——后面的原则不知道
