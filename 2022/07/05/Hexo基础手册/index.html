Fornt-Matter

参考Hexo官方文档

Fornt-Matter听起来高大上，实际上便是在Markdown编辑文章中的头文件，用于给该文章标注信息

示例

title: { { title } }
date: { { date } }
updated:
tags:
categories:
thumbnail:
top: false
comment:

Title

文章的标题
使用hexo n"{ {title} }"创建文章时会自动将{title}填入title中

Date

文章的创建日期

Update

文章的更新日期

Tags

文章标签

tags:
- Blog
- Hexo

Categories

文章分类 

categories:
- [笔记, Hexo]
- [博客]

即该文章归类为两个分类：笔记和博客，Hexo为笔记的子分类

自己的习惯还是一个分类 多个标签

Thumbnail

部分Hexo主题拥有的功能，文章缩略图 （封面）

首先需要在根目录/source新建一个名为thumbnail的文件夹 将图片存放至该文件夹中

然后在Front-Matter中引用照片

thumbnail: /thumbnail/1.jpg

Top

Hexo插件功能 将你的某一文章置顶

默认toc: false 需要置顶时将false改为true即可

Comment

评论功能 需要运用第三方插件 现博客未应用

Hexo基本指令

在站点根目录右键Git Bash Here进入git终端

Clean

$ hexo clean

清除缓存文件和已经生成的静态文件 在更改主题设置时需要清理后更改才能生效

Generate

$ hexo g

生成静态文件（就是生成一个博客）

Sever

$ hexo s

启动服务器（就是将你的博客先部署到本地生成的一个页面）默认网址为http://localhost:4000/

停止服务器

$ ctrl+ c

Deploy

$ hexo d

部署网站

一般发布博客的顺序为：

$ hexo clean
$ hexo g

然后先在本地localhost:4000查看效果

$ hexo s
$ ctrl+ c

最后部署到GitHub服务器上

$ hexo d
